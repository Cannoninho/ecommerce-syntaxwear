
# SyntaxWear

SyntaxWear é um site de e-commerce de calçados fictício, criado como parte de um projeto de desenvolvimento web. O site apresenta um design moderno e responsivo, com foco na venda de tênis e sneakers.

## Visão Geral

O projeto consiste em uma única página (index.html) que exibe os produtos da loja de forma atraente. A página é estilizada com CSS e dividida em vários componentes, como cabeçalho, herói, grade de produtos e rodapé.

## Tecnologias Utilizadas

* **HTML5:** Para a estrutura da página.
* **CSS3:** Para a estilização e o layout.

## Funcionalidades

* **Layout Responsivo:** O site se adapta a diferentes tamanhos de tela, proporcionando uma boa experiência de usuário em desktops, tablets e smartphones.
* **Navegação:** O cabeçalho inclui um menu de navegação com links para as seções do site.
* **Destaques:** A seção "hero" apresenta um produto em destaque com um call-to-action.
* **Categorias:** Os produtos são organizados em categorias, como "Casual", "Esporte", "Moderno" e "Futurista".
* **Grade de Produtos:** Uma grade de produtos exibe os itens da loja de forma visualmente atraente.
* **Rodapé:** O rodapé contém links úteis, informações de contato e um formulário de inscrição para a newsletter.

## Como Executar o Projeto

Para visualizar o site, basta abrir o arquivo `index.html` em seu navegador de preferência.

## Personalização

O código-fonte é bem organizado e comentado, facilitando a personalização. Você pode facilmente alterar as cores, fontes, imagens e produtos editando os arquivos CSS e HTML.
